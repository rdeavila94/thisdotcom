I specialize in web application development with Java and JavaScript

I use a typical enterprise Java development stack involving: Maven, Spring, Hibernate, Oracle (or any major database, depending on how the data access layer is abstracted), and Thymeleaf (or JSP, if necessary).
Full-stack development with JavaScript is accomplished — obviously — with Node.js, particularly using the wittily dubbed "MEVN" stack, in contrast to the MEAN stack. For front-end development my orientation is towards Vue out of the many JS frameworks although Vanilla JS, the framework that never was, is held dear to my heart. Regarding styling, I have an affinity towards Sass and prefer it over plain CSS. Of course, I also have Bootstrap in my toolkit for that domain.
Outside of the aforementioned tech, I also like to dabble in functional programming with Elixir and desktop development with Electron.

This.com

This website was built to 1) establish an online presence and 2) return to the fundamentals of front-end dev. I wrote every line of HTML, SCSS, and JavaScript; no use of frameworks or libraries whatsoever. Compilation tools were obviously involved to transpile ES2015+ JavaScript to older JavaScript for browser compatibility, and to compile SCSS to CSS. The artistic approach was for the site be lighthearted and risky.
