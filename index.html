Identicon

A small app I created that uses an Elixir back-end to turn text into an identicon. This was my first real project using functional programming, broadening how I approach software development since I come, like many others, from an OOP background. Of particular interest to me was the structure of the Phoenix web framework; learning it was a completely separate journey of its own. A simple front-end that makes ajax requests to the server was paired to showcase the back-end.

This.com

This website was built to 1) establish an online presence and 2) return to the fundamentals of front-end dev. I wrote every line of HTML, SCSS, and JavaScript; no use of frameworks or libraries whatsoever (VanillaJS isn't a framework, fyi). Compilation tools were obviously involved to transpile ES2015+ JavaScript to older JavaScript for browser compatibility, and to compile SCSS to CSS. The artistic approach was for the site be lighthearted and risky.
